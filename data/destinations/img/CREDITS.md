# Image credits

One photo per registry destination, filename = lowercase destination name + `.jpg`.
These files are committed and shipped with the public deploy, so every one of them
**must be free to redistribute: CC0 / public domain only** (the licensing decision is
locked in `PLAN-v2.md`, idea 4). Record the source when a file lands — an image with
no row here shouldn't be merged. Keep files web-sized (≲ 500 KB) — they live in git.
